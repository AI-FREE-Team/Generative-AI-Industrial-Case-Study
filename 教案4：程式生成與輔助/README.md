# 程式生成與輔助
日常生活中隨處可見的 3C 設備，每天都在用的手機 APP 或是電腦程式系統或是網頁介面，裏頭都有著無數的程式自動化的運作著。在生成式 AI 崛起之前，這些程式碼都是由無數軟體工程師撰寫而成；而在生程式 AI 出現之後，越來越多系統背後的程式碼將會由生成式 AI 輔助軟體工程師撰寫而成。
