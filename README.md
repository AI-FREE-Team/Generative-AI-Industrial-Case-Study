# 生成式AI於產業應用教案實作練習模組
此開源教案針對不同產業面向提出5個生成式AI工具的實務應用情境，期待能夠幫助中文使用者在生成式AI工具崛起的時代中，順利掌握相關的工具，，並進一步加值於自身的產業領域當中。
